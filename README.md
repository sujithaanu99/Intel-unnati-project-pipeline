## Project Title
Scalable Object Detection and Classification Pipeline using OpenVINO and DL Streamer

## Description
A real-time multi-stream video analytics pipeline built using Intel OpenVINO and DL Streamer.
The system performs:
- Person and vehicle detection
- Attribute classification (age, gender, vehicle type, color)
- Performance analysis using FPS and latency metrics

## Technologies Used
- Python
- OpenVINO Toolkit
- DL Streamer
- GStreamer
- OpenCV
